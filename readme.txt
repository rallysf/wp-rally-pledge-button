Embed a rally.org pledge widget in your wordpress blog

== Description ==

Provides a button in post editor for adding a shortcode that links to a
rally on rally.org

= Use it =

A shortcode is available to be used in your posts or pages :

* [rally-pledge page="http://rally.org/buzkashiboys"] --> embed the
  cover page widget block for http://rally.org/buzkashiboys
